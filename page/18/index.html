svn add file\
例如：svn add test.php(添加test.php)\

\

简写：svn ci\
\

例如：svn lock -m “lock test file“ test.php\
svn unlock PATH\

vi格式化代码

有些时候复制别人的代码会非常乱，因此格式化代码就很重要了\
vi 下格式化代码：\
1.全部代码格式化：\
命令行模式下\
按下 gg （回到开头）\
按下 = （格式化命令）\
按下 G （代码尾部）\
2.部分代码格式化：\
命令行下\
选中你要格式化的代码\
按下 =\
3.\
(1) 按两下小写g，即gg，定位光标到第一行。\
(2) 按住Shift+v，即大写V，进入可视化编辑的列编辑模式。\
(3) Shift+g，即大写G，选中整个代码。\
(4) 按下等号=，格式化所有代码。
