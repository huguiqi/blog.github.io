  1. Hexo在github上构建免费的Web应用

不过在做那些操作时，你的电脑需要这些环境：

  1. nodejs
  2. ruby

  3. 浏览器
    由于我的电脑是mac，我本人又是一枚程序猿，所有这些环境都有，如果有人不知道怎么安装这些环境的:

    brew install nodejs

安装hexo库

hexo的官网

如下方式安装：

$ npm install hexo-cli -g
$ hexo init blog
$ cd blog
$ npm install
$ hexo server
