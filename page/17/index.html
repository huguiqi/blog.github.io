MAC OS X 关闭 SPOTLIGHT MAC降温大法

By sam hu

发表于 2016-03-30

MAC OS X 关闭 SPOTLIGHT MAC降温大法

MAC OS X 关闭 SPOTLIGHT MAC降温大法\

禁用分两部分，第一部分禁用服务，第二部分移除菜单栏图标，以下动作需在终端中进行\
禁用方法 1，禁用spotlight服务\
sudo mdutil -a -i off\
2，删除菜单栏spotlight图标：\
cd /System/Library/CoreServices/ sudo mv Search.bundle/
Search2.bundle/\
然后在活动监视器退出 SystemUIServer 进程，或者注销再登陆。\
恢复方法 1，启用spotlight服务，\
sudo mdutil -a -i off\
2，恢复菜单栏图标\
cd /System/Library/CoreServices/ sudo mv Search2.bundle/
Search.bundle/\
然后在活动监视器退出 SystemUIServer 进程，或者注销再登陆。\
spotlight的替代品：
一般我们用spotlight主要是干两件事，第一是快速启动程序，第二是快速查找文件，对于前者，lion中的lauchpad已经很好用了，建议自己好
好整理一下，将常用的程序放在前面，不常用的按照种类文件夹管理，这是个一劳永逸的方法，速度不会比spotlight慢，如果愿意研究，可以试试
QuickSilver或者alfred之类的启动器软件，稍微复杂些；对于后者，我选择了一个叫easyfind的软件，它是实时搜索的，不用建立索
引，由于air硬盘小，所以文件也不少，加上SSD也不慢，所以即用即搜也ok了。

spring3.0发送电子邮件（velocity模板，带附件，群发，解决乱码



spring3.0发送电子邮件（velocity模板，带附件，群发，解决乱码



\



\



在这个例子中，将与发送方相关的配置信息放在了一个email.properties文件中，spring容器启动的时候会从这个属性文件中读取发送方的配置信息，这样配置的主要原因在于，发送方一般都是固定不变的。



我将发送的内容放在一个velocity模板文件中，这个文件很像jsp文件，您可以从http://airport.iteye.com/blog/23634中获取更多的关于velocity使用方法相关的信息。（实际上velocity是Apache提供的一种模板语言）。我希望邮件接收方能够接收一个动态的网页，而不是一个简单的文本。



另外，程序提供了发送附件的部分，和群发的功能，只不过群发功能并不完善，因为群发过程中无法将模板变量替换成用户相关变量值，这是个缺陷，楼主也没想明白如何实现。
