synthesize和 dynamic区别

synthesize和 dynamic区别\ \
在声明property属性后，有2种实现选择\
synthesize\
编译器期间，让编译器自动生成getter/setter方法。\
当有自定义的存或取方法时，自定义会屏蔽自动生成该方法\
\
dynamic\
告诉编译器，不自动生成getter/setter方法，避免编译期间产生警告\
然后由自己实现存取方法\

或存取方法在运行时动态创建绑定：主要使用在CoreData的实现NSManagedObject子类时使用，由Core
Data框架在程序运行的时动态生成子类属性

不过在做那些操作时，你的电脑需要这些环境：

  1. nodejs
  2. ruby

  3. 浏览器
    由于我的电脑是mac，我本人又是一枚程序猿，所有这些环境都有，如果有人不知道怎么安装这些环境的:

    brew install nodejs

安装hexo库

hexo的官网

如下方式安装：

$ npm install hexo-cli -g
$ hexo init blog
$ cd blog
$ npm install
$ hexo server
