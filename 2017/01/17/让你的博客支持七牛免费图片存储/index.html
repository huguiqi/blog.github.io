让你的博客支持七牛免费图片存储

hexo-admin

hexo有一个后台管理系统的插件— ‘hexo-admin’，但是一直以来不太好用，基本上我都是用mou,一直以来都挻喜欢mou的，但是由于mac系统更新到10.12之后，mou就打不开了，而mou已经停止更新了，目前我用的是UIysses,这个软件是收费的，但在咱天朝，破解版本还是挻好找的；
但无论是用mou，还是UIysses，还是markdown的其它工具，写文章对于图片来说，都不太方便，特别是对于截屏来说，不能即截即用；

hexo-admin-qiniu

前一阵子在hexo-admin的github上提交了一个issue,呼吁增加hexo-admin手动发布和发布文章可以直接粘贴图片的问题，16年年底有人在github回应了我，提交了hexo-admin-qiniu版本，好开心^&^;
这是github地址：
https://github.com/xbotao/hexo-admin-qiniu
按照上面的方法，安装了hexo-admin-qiniu插件，然后：

hexo server -d

浏览器地址栏输入：
http://localhost:4000/admin
进入文章页面，点编辑，然后在内容里，随便截图一张，copy到这里，然后就ok了，点publish；

再到发布页面点击deploy按钮；
和命令行：

hexo g && hexo deploy // 是一样的效果
